# Welcome

Hey there! Welcome back (Unless it's your first time visiting, then it's just welcome) to the Dreadwinter website!

The site was previously hosted on Wix, but I didn't feel like it was worth paying the yearly subscription, so I didn't renew it at the end of last year, which meant that it was hidden for some time.

I decided instead to find a free option, and so it's now hosted using Github Pages! It took a while to setup up, since I had to do a lot of the work that a Content Management System usually handles for you. The actual setup wasn't really that long, but I put it off for a while...

Currently, my monthly devlogs are posted on the [Dreadwinter Steam community hub](https://steamcommunity.com/app/2505230), and I may post them here as well. Otherwise, I think it would be fun to use this space as a personal blog, and post other things that aren't directly related to Dreadwinter, like game dev advice, or just what I've been playing.

As a little update, Dreadwinter is still plodding along! I think the target release date will be a bit later than initially planned (it was intended to be in the next month or two), since there's been some things going on, and my health wasn't in the best spot last month. Still, a little bit of progress is better than no progress, so it's not all bad!

That's it for this post, thanks for reading, and I'll see you in the next post
-Alex
